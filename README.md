GitHub Profile Analyzer

- Project Overview

GitHub Profile Analyzer is a web application that analyzes a GitHub user's profile and displays useful statistics using the GitHub API.
The application provides insights such as repository information, language usage, contribution activity, and developer score.

This project demonstrates API integration, dynamic data visualization, and modern frontend development using HTML, CSS, and JavaScript.

- Features

* Search any GitHub username
* Display GitHub profile information
* Show repository names and creation dates
* Display repository stars
* Language usage chart
* Contribution heatmap
* Developer score calculation
* Repository search filter
* Dark mode toggle
* Direct link to GitHub profile and repositories

- Technologies Used

* HTML
* CSS
* JavaScript
* GitHub REST API
* Chart.js (for data visualization)

- How to Run the Project

1. Clone the repository

2. Open the project folder in VS Code.

3. Open **index.html** in a browser
   or use **Live Server extension** in VS Code.

4. Enter a GitHub username and view the analytics.

- Data Source

This project uses the **GitHub REST API** to fetch public user profile data and repository information.
